User Manual & Operations

This manual outlines the operational steps to organize your local folders using SortMind AI.

Scanning a Directory

1. Launch the SortMind AI application on your desktop.
2. Click the 📁 Browse button next to the "Target Folder" input. This initiates your operating system's native folder browser dialog.
3. Select the cluttered folder and click "Select Folder".
4. Click the Scan Directory button.

The application scans all files immediately inside that folder and renders them inside the files table. Nested subdirectories are bypassed for safety.

Dry Run vs Organize Execution

Choose the specific files you want to organize by checking the boxes in the leftmost column. Select one of the processing actions:

  • AI Preview (Dry Run): Extracts content snippets, sends them to the configured AI engine, and returns predictions in the "Category" column. No files are moved on your disk. Use this mode to evaluate categorizations before execution.
  • Organize Files: Classifies selected files, automatically creates target category directories within the scanned folder, and moves the files.

Run-Time Controls (Pause & Cancel)

During folder organization, SortMind AI processes your files sequentially. You can control the active run with the following control buttons:

  • Pause: Temporarily halts the sequence. The active file finishes processing, and the queue waits. The button will toggle to "Resume". Click "Resume" to continue.
  • Cancel: Immediately terminates the remaining queue after the current active file finishes processing. Control binds are safely restored.

Filename Collision Policy

If a file being moved has the exact same name as a file already present in the target category folder, SortMind AI prevents overwriting.

It automatically appends an incremental numeric suffix to the filename before completing the move (e.g., if invoice.pdf already exists inside the Invoices folder, the incoming file will be renamed and moved as invoice_1.pdf).
