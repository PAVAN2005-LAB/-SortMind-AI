Windows Installation

Follow these instructions to run the application on Microsoft Windows systems.

1
Extract ZIP:

Right-click the downloaded SortMindAI_Windows_x64.zip file and extract its contents to an directory (e.g., Desktop or Program Files).

2
Run Installer:

Open the extracted folder and double-click SortMindAI_Setup.exe to start the setup wizard.

3
Accept License:

Read the presented Terms and Conditions. Click the acceptance check box to agree and enable the installation progression.

4
Finish Installation:

Select whether to generate a desktop shortcut, click Install, and launch the native application window.

System Prerequisite: The application relies on the Microsoft Edge WebView2 Runtime to display its user interface. WebView2 is pre-installed on Windows 10 and 11 by default. No manual runtime installations are needed.

Linux Installation

Follow these instructions to run the application on Linux distributions (Ubuntu, Debian, etc.).

1
Install WebKit2Gtk Dependency:

Linux requires the WebKit2Gtk library to render the application window. Install it using the package manager in your terminal:

sudo apt update && sudo apt install libwebkit2gtk-4.0-dev -y
2
Extract and Run Binary:

Extract the tar package, navigate to the directory, mark the binary executable, and run it:

tar -xzf SortMindAI_Linux_x64.tar.gz
chmod +x SortMindAI
./SortMindAI
