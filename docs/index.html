SortMind AI

SortMind AI is a fast, native desktop application that automates folder organization. Instead of sorting files simply by extension, SortMind AI reads file content snippets or metadata, asks a local or free AI API what the file actually represents, and automatically moves it into context-aware subfolders.

Note: SortMind AI runs entirely locally. It does not run background server processes or store your personal documents. All API keys and preferences are stored securely in a local configuration file on your machine.

Key Principles

  • Content-Aware Classification: Uses advanced LLM processing to analyze snippets and match file topics dynamically rather than checking extensions.
  • Offline-First Capabilities: Integrates with local Ollama runtimes so you can organize folders privately without sending details to cloud APIs.
  • Responsive Local Framework: Developed using the Go + Wails v2 desktop stack to compile into a single fast binary (no bulky browser wrappers).
