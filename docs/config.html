AI Engine Setup

Before classifying files, you need to configure your chosen AI provider in the application settings panel.

Provider Supported Models Credentials Offline Capable
Google Gemini Gemini 2.5 Flash, Gemini 2.5 Pro Requires API Key from Google AI Studio. No (Cloud API)
OpenAI GPT-4o Mini, GPT-4o Requires API Key from OpenAI Developer Platform. No (Cloud API)
Ollama Llama 3, Mistral, custom local models None (connects to local port 11434 automatically). Yes (100% Offline)

Google Gemini Configuration

Gemini 2.5 Flash is recommended due to its high classification speed and free tier options. Visit the Google AI Studio API Key portal, copy your key, select "Google Gemini" in the dropdown, paste the key, and click "Save Settings".

Ollama Offline Setup

For offline security and private sorting, install Ollama from ollama.com. Open your command terminal and download a compatible text classification model:

ollama pull llama3

In the SortMind AI dropdown, choose "Ollama", select "Llama 3" in the model dropdown, and click "Save Settings". No API key is required.

Custom Folder Categories

The category chips listed in the sidebar represent the target directories files will be sorted into. You can custom tailor this list:

  • Add Category: Enter the name of the new target folder in the "New folder name" input field and click the "+" button. The name will appear as a category chip.
  • Delete Category: Click the "x" button next to any category chip to remove it.
  • Save Settings: Click the Save Settings button to write these custom categories to the configuration file.
Default Categories: If no custom categories are specified, SortMind AI defaults to: Invoices, Receipts, Readings, Code, Images, and Others.
